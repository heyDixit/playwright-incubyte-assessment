# Playwright BDD Test Suite

## Overview

This is a Playwright BDD framework created for the Incubyte assessment. It's a behavior-driven development test automation framework built with Playwright and Cucumber.

## Quick Start

1. **Clone the repository to local**

2. **Install all dependencies**

   ```bash
   npm i
   ```

3. **Create `.env` file** (Optional)
   Create a `.env` file in the root directory and add an existing username:

   ```
   USER=yourexistingusername
   ```

4. **Run the tests**
   ```bash
   npm test
   ```
